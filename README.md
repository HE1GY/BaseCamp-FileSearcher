# FileSearcher

Search starts from root. (Linux "/", Windows "C:\\")

Build command:

```bash
make
```

Run command:

```bash
cd build
```

- Linux:

```bash
./FileSearcherApp FILE_NAME_WITH_EXTENSION
```

- Windows(MINGW64):

```MINGW64
./FileSearcherApp.exe FILE_NAME_WITH_EXTENSION
```

## Warning

**App is sensitive to recursive symbolic links !!!**
